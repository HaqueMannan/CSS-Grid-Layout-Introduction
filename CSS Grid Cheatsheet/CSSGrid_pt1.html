Lesson 1

Create a div class wrapper/container (or whatever you would like to call it) - this will act as the wrapper/container for the CSS grid. Elements within the container will have the CSS Grid Layout

We have added a wrapper-format to visually represent the Grid layout of the content below.

To add the CSS Grid layout to the wrapper class we must add in the CSS display:grid; as you would do in flexbox by adding display:flex.

Next we would need to define the width of our grid columns using the grid-template-columns: property. In the example we gave a 70% 30% grid layout for the wrapper class. Therefore, no matter how many div elements we have within the wrapper class they will all continue to follow the 70:30 column grid (as seen below). We do not need to define the rows unlike bootstrap (another CSS framework created by the Twitter Team).

To add spacing between the columns we can add the grid-column-gap: property. To add spacing between the rows we can add the grid-row-gap: property. To add spacing between both the columns and rows we can add the grid-gap: property. This saves us from manually adding a margins, also note that you can use any other units such as px, cm, % etc.
