Lesson 2

There is a new unit introduced by CSS-Grid called fractionals (fr) which is an alternative to percentage.

It is recommended to use fractionals over percentages for the CSS grid units because when you use percentages you will end up running into issues with margins and paddings.

We can use the repeat(); function to shorten our syntax codes for example:
grid-template-columns: 1fr 1fr 1fr; can also be written as grid-template-column: repeat(3, 1fr);. The repeat function takes in two parameters, the first is the number of repeat and the second is the unit measeurement.
The shorthand syntax of grid-template-column: repeat(2, 1fr 2 fr); is the same as writing the longhand syntax of grid-template-columns: 1fr 2fr 1fr 2fr; (the column grid 1fr 2fr is repeated twice to make a 4 column grid).

To fix the height of all column rows we would use grid-auto-rows: property. If the content of a div is more than the row height then all the contents of the div will fall outside the row height.

To make the row height flexible we can use the function minmax(). The two parameters for the minmax() function are the minimum height (default) and the maximum height.
For example: minmax(100px, auto) would format the CSS grid to have a default grid height of 100px and a maximum height of auto. This would therefore allow the grid to be more flexible and increase its height for the particular row should a div element content go beyond the 100px default height.
