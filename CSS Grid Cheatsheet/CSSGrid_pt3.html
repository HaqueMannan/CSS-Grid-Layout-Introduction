Lesson 3

We can justify column grids left, center and right by using the justify-items: property.
justify-items: start; will justify the content to the left, justify-items: center; will justify the content to the center and justify-items: end; will justify the content to the right. justify-items: stretch; is the default.

We can also align column grids top, center and bottom by using the align-items: property.
align-items: start; will align the content to the top, align-items: center; will align the content to the center and align-items: end; will align the content to the bottom. align-items: stretch; is the default.

We can also use align-self: and/or justify-self: (the same as above) but this will allow each div element to have its own independant justify/align property whereas the above would apply to all elements.


The grid-column: and grid-row: properties allows elements to span across columns and rows which allows CSS Grid to be two-dimentional. This means elements can overlap each other or certain elements to take up a certain amount of column/row space on a webpage. The numbers in the parameters relate to the line number of the grid-column/grid-row. In the example below:
grid-column:1/3; for .box1 div class will expand its content in 2 columns and grid-row:1/3; and two rows.
A three column grid will have 4 lines to make up is columns. Line 1 and 2 will create the first column, line 2 and 3 will create the second column and linke 3 and 4 will create the third column (same ruling applies to the grid rows lines). Therefore when adding the parameters to the grid-column: property, we are referencing the grid lines for the element column.

With grid-columns we have the flexibility/power to rearrange items and further more the ability to overlap items (which is not possible in flexbox).
